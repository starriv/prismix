# Prismix Engineering Tasks

This directory tracks multi-step engineering work that needs durable context beyond a chat thread.

- `index.md` is the dashboard.
- `inbox.md` captures ideas or risks that are not formal tasks yet.
- `projects/` holds project-level context and milestone state.
- `tasks/` holds executable task cards with status, dependencies, acceptance criteria, logs, and review notes.

Task status values: `todo`, `doing`, `blocked`, `done`.
