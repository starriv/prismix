# Git Rules

**Squash merge only** to `main`. No regular merge, no rebase merge.

Commit message: `<type>(<scope>): <summary>` + bullet body (max 10) + `Co-Authored-By: Claude Opus 4.6 <noreply@anthropic.com>`.
Types: feat/fix/refactor/style/docs/test/chore. Scope optional (ai/web/server/db). Summary: English, lowercase, no period.
